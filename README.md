# PRECO

Library for Predictive Coding Networks (PCNs) and Predictive Coding Graphs (PCGs) in Python, following submission accepted at **ACM Computing Surveys**.

**Title:** Predictive Coding Networks and Inference Learning: Tutorial and Survey  
**Authors:** Björn van Zwol, Ro Jefferson, Egon L. van den Broek  
**URL:** [doi/10.1145/3797870](https://dl.acm.org/doi/10.1145/3797870) 

Uses PyTorch for GPU accelaration and familiar PyTorch interfaces.
